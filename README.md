# ADS-B SDR Local

Local ADS‑B integration for Home Assistant using readsb or dump1090.

## Features
- Aircraft count
- Nearest aircraft distance
- Nearest callsign
- Lowest altitude

## Default endpoint
http://<receiver_ip>:8080/data/aircraft.json

## Install via HACS
Add this repository as a Custom Repository (Integration):
